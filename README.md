# kotlin-lotto-precourse

## 입력/출력
### 입력
#### 당첨 번호 입력 받기
- 숫자 6개 입력
-> 6개 아니면 에러, 이것도 범위는 1~45로 입력해야 함

#### 보너스 번호 입력 받기
-  숫자 1개 입력
-> 당첨 번호랑 겹치지 않는 번호로 입력, 숫자 하나 입력, 이것도 범위는 1~45

#### 구입 금액 입력 받기
- 로또 금액은 1000원
-> 1000원보다 낮은 금액 입력 시 에러
- 입력한 금액에서 1000을 나눈 값이 구매한 총 금액의 개수
-> 1000으로 나눠 떨어지지 않아도 그냥 진행 (5500원 입력 시 5개 구매로 간주)

### 출력
- 각 문구나 로또 번호, 통계 등 출력

## 로또 번호 생성 
### 로또 번호 추출하기
#### 구매한 로또 개수만큼 Random 숫자 6개씩 추출
-> 범위는 1~45

## 로또 결과 생성
### 당첨 통계 추출하기
#### 일단 몇 등인지 확인
#### 당첨된 금액이 있으면 이익 계산
