# 로또

## 기능 목록

### 입력
- [ ] 로또 구입 금액 입력
- [ ] 당첨 번호 입력 (쉼표로 구분된 숫자 6개)
- [ ] 보너스 번호 입력 (숫자 1개)

### 로또 발행
- [ ] 구입 금액에 해당하는 로또 수량 계산 (1,000원 당 1장)
- [ ] 중복 없는 6개의 로또 번호 랜덤 생성 (1~45 범위)
- [ ] 발행한 로또 번호 오름차순 출력

### 당첨 확인
- [ ] 구매한 로또 번호와 당첨 번호 비교
- [ ] 보너스 번호 일치 여부 확인 (2등 판별)
- [ ] 일치 개수에 따른 등수 판별 (1~5등)

### 결과 출력
- [ ] 등수별 당첨 내역 출력
- [ ] 수익률 계산 (소수점 둘째 자리에서 반올림)
- [ ] 수익률 출력

### 예외 처리
- [ ] 구입 금액 유효성 검사 (양의 정수, 1,000원 단위)
- [ ] 당첨 번호 유효성 검사 (1~45 범위, 중복 없음, 6개)
- [ ] 보너스 번호 유효성 검사 (1~45 범위, 당첨 번호와 중복 없음)
- [ ] 잘못된 입력 시 `[ERROR]`로 시작하는 메시지 출력 후 해당 입력 재요청


## 실행 방법

**요구 사항:** Java 21

```bash
# 테스트 실행
./gradlew test

# 애플리케이션 실행
./gradlew run
```
